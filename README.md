# 플레이리스트

저희 팀이 만든 홈페이지의 주제는 플레이리스트입니다.
밤과 새벽, 위로 받고 싶을 때 들을 수 있는 라디오 감성 명곡들을 추천해주는 플레이리스트를 만들어 보았습니다.

힘들거나 지칠 때 위로를 건네주는 라디오 DJ 같은 곡, 엔플라잉의 songbird

마음을 편안하게 해주고 담담한 목소리로 감동을 주는 곡, 아이유의 밤편지

겨울과 잘 어울리는 따뜻하고 다정한 곡, 최립우의 My Friend

홈페이지에서는 이렇게 총 세곡으로 각자가 추천하는 곡들이 무엇인지 볼 수 있고, 어떤 곡들인지에 대한 설명, 이 곡들을 추천하게된 배경,이유를 읽어볼 수 있습니다. 그리고 곡들을 바로 들어볼 수 있도록 유튜브에서 듣기를 누르면 유튜브와 연결되어 곡들을 들어볼 수 있게 만들었습니다.

### 역할분담

김래현(조장)-메인저장소 제작, 브랜치 제작, html/stlye파일 제작, html파일에 코드 추가/수정/커밋, 풀리퀘스트 리뷰어, 프로젝트/이슈 작성, 리드미파일 작성.

김서현-브랜치 제작, html/stlye파일 제작, html파일에 코드 추가/수정/커밋, 풀리퀘스트 리뷰어.

김보민-브랜치 제작, html/style파일 제작, html파일에 코드 추가/수정/커밋, 풀리퀘스트 리뷰어.

처음에는 각자 브랜치를 만들고 html/stlye파일을 만들어 본인의 코드를 추가시킨 후 메인저장소에 병합을 진행했는데 그 결과 오류가 생기고 원하는 홈페이지의 형태가 나오지 않았습니다. 그래서 전면적인 수정을 거쳐 한 사람이 html/stlye파일을 만들고 본인의 코드를 추가시키면 그 다음 사람이 그 파일을 받아 본인의 코드를 추가시키고 수정하여 커밋해나가는 방식으로 진행하여 플레이리스트 홈페이지를 완성했습니다.
